# Docs Icons

NPM icon packages providing uniform Iconify JSON icon sets with Astro components for the F5 XC documentation build system.

## Icon Reference

- [Overview](/icons/) — Available icon packs, rendering methods, and selection guide
- [Starlight Built-in](/icons/starlight/) — 200+ built-in icons via the `&lt;Icon&gt;` component
- [F5 Brand](/icons/brand/) — 666 SVG icons for networking, security, cloud, and AI
- [Lucide](/icons/lucide/) — 1,600+ modern stroke icons
- [Iconify Packs](/icons/iconify-packs/) — Material Design, Carbon, Phosphor, and Tabler
- [HashiCorp Flight](/icons/hashicorp-flight/) — 671 cloud and infrastructure vendor icons
